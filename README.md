# Netflix Data Analysis

This project analyzes Netflix movies and TV shows dataset using Python.

## Dataset
The dataset was obtained from Zenodo.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Movies vs TV Shows distribution
- Content released over the years
- Top countries producing Netflix content
- Rating distribution
- Genre analysis
